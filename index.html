MICROSOFT STOCK PRICES PREDICTION¶

Introduction¶

For this project, we will be making use of Python and a machine learning technique called Long Short-Term Memory (LSTM) with the Yahoo Finance dataset.

LSTMs are widely used for sequence prediction problems and have proven to be extremely effective.The reason they work so well is because LSTM is able to store past information that is important, and forget the information that is not. -Stackabuse.com

It must be noted that program will be limited on making use of historical data only to create a prediction of the stock prices. It should only provide an idea of the trend but stock prices in general are affected by many things such as the macroeconomy, investors, political climate, etc.

# Create training data set where the past 60 days closing prices values will be used to predict 61st closing price value.
# past 60 days: 'x_train 1st col with values of index 0 to 59 (60 values total) and 2nd col with values of index 1 to 60...
# target label: y_train 1st col to contain the 61st value located at index 60 and the 62nd value located at index 61...

microsoft_quote = web.DataReader('MSFT', data_source='yahoo', start='2015-01-01', end='2020-07-27')

#Create a new dataframe
new_df = microsoft_quote.filter(['Close'])

#Get the last 60 days closing price 
last_60_days = new_df[-60:].values

last_60_days_scaled = scaler.transform(last_60_days)

#Create an empty list
X_test = []

# Append the past 60 days

# Convert the X_test data set to a numpy array

X_test = np.reshape(X_test, (X_test.shape[0], X_test.shape[1], 1))

# Get the predicted scaled price
pred_price = model.predict(X_test)

# Undo the scaling 
pred_price = scaler.inverse_transform(pred_price)
